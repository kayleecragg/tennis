Instructions for Chrome on Mac:

  1. Click "Request Permission" - you'll see a browser permission popup
  2. Select "Allow" to enable desktop notifications
  3. Notifications will automatically start every 30 seconds
  4. Use the "Stop Notifications" button to cancel the timer
  5. If you accidentally block notifications, refresh the page and try again
